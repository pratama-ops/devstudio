# 🚀 DevStudio Landing Page

Landing page modern untuk jasa pembuatan website (agency-style) dengan desain clean, responsif, dan fokus konversi.

## ✨ Fitur Utama

* 🎯 Hero section dengan CTA jelas
* 🧩 Section layanan (Landing Page, Company Profile, dll)
* 💰 Pricing tab interaktif
* ⚙️ Proses kerja (step-by-step)
* ⭐ Testimoni klien
* 📱 Fully responsive (mobile-friendly)
* 🎨 UI modern (dark theme + gradient accent)
* 🧭 Smooth scroll navigation
* 🔤 Typography optimized (tidak gepeng & lebih readable)

---

## 🛠️ Teknologi yang Digunakan

* HTML5
* CSS3 (Custom styling, no framework)
* Vanilla JavaScript
* Google Fonts (Syne & DM Sans)

---

## 📂 Struktur Section

* Hero
* Layanan
* Harga (tab system)
* Proses
* Testimoni
* CTA
* Footer
